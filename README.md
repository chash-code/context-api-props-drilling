# context-api-props-drilling
# Solving Props Drilling Using Context API

This project demonstrates how to use React Context API to eliminate props drilling by providing shared data to deeply nested components without manually passing props at every level.

## Project Structure
├── src/
│   ├── App.js
│   ├── AppContext.js
│   ├── Component1.js
│   ├── Component2.js
│   ├── Component3.js
│   ├── Component4.js
│   ├── Component5.js
│   └── Component6.js
├── package.json
└── README.md
## Features

- Context API implementation
- No props drilling
- Clean component structure
- Values displayed using `<h4>` tags
